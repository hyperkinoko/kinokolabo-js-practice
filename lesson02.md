## 課題4

定数numを宣言し、0から99のランダムな数字で初期化します。

テンプレートリテラルを使って、

```
56は偶数です
```

のように表示しましょう。

###  ヒント

0以上x未満のランダムな数字を生成し、定数numに代入するコードは、

```javascript
const num = Math.floor(Math.random() * x);
```
です。

```

```

```
5は奇数です
```

のように表示しましょう。

### ヒント

1〜6のランダムな数字を取得するには、  
まず0〜5のランダムな数字を取得し、それに1を足すといいでしょう。
